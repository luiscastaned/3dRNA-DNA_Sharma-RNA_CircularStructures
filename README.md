## Descripción
Este repositorio contiene los modelos 3D de estructuras terciarias de RNA/DNA predichos utilizando el servidor **3dRNA/DNA** del Xiao Lab, para dos secuencias circulares.

## Secuencias Analizadas
1. **EHI_137090** - Estructura circular
2. **EHI_016130** - Estructura circular

## Métodos de Predicción
Para cada secuencia se generaron dos tipos de modelos:
- **Modelo Default**: Predicción con parámetros por defecto
- **Modelo Optimized**: Predicción con parámetros optimizados

## Contenido del Repositorio
- `/EHI_137090/` - Modelos para la secuencia EHI_137090
- `/EHI_016130/` - Modelos para la secuencia EHI_016130
- `Results_Summary.pdf` - Resumen de los resultados
- `Experimental_Conditions.pdf` - Condiciones experimentales y parámetros utilizados

## Herramienta Utilizada
- **3dRNA/DNA Server**: [http://biophy.hust.edu.cn/new/3dRNA](http://biophy.hust.edu.cn/new/3dRNA)
- Método de predicción de estructuras terciarias de RNA y DNA desarrollado por Xiao Lab

## Referencia del Método
3dRNA/DNA: An RNA and DNA tertiary structure prediction method

## Cómo Usar los Archivos
Los archivos PDB pueden visualizarse con:
- PyMOL
- UCSF Chimera
- VMD
- Jmol
